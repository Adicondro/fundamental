Understanding HTML

HTML (HyperText Markup Language) is the standard language for creating web pages. It describes the structure of a web page and its content.

Basic HTML Structure

Every HTML document starts with a <!DOCTYPE html> declaration, followed by <html>, <head>, and <body> tags.

Elements, Tags, and Attributes

HTML elements are defined by tags. Some common tags include:

  • <h1> - Heading 1
  • <p> - Paragraph
  • <div> - Division or section
  • <a> - Anchor (link)
  • <img> - Image
  • <span> - Inline section

Basic Text Formatting

HTML provides several tags for text formatting:

  • <strong> - Bold text (strong importance)
  • <em> - Italic text (emphasis)
  • <b> - Bold text
  • <i> - Italic text
  • <u> - Underlined text
  • <mark> - Highlighted text
